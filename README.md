## Installation

Via [Vundle](https://github.com/VundleVim/Vundle.vim) (recommended):
```
Plugin 'jameslawson/sandwich.vim'
```

## Development

You'll need to have a Ruby environment setup.

In order to run the **unit tests**:
```
bundle install
rake test
```
